Overview.

The definitions described in this section are used at ECMWF for special purposes.

Element KSEC1(24) is set to 1 if an ECMWF local use definition is required or has been found in a GRIB message; element KSEC1(37) indicates which definition it is.

If KSEC1(24) is set to 0, no local use definition is present in the message.

Currently, the following definitions exist:

  • Definition 1 - MARS labelling or ensemble forecast data
  • Definition 2 - Cluster means and standard deviations
  • Definition 3 - Satellite image data
  • Definition 4 - Ocean model data
  • Definition 5 - Forecast probability data
  • Definition 6 - Surface temperature data
  • Definition 7 - Sensitivity data
  • Definition 8 - ECMWF re-analysis data
  • Definition 9 - Singular vectors and ensemble perturbations
  • Definition 10 - EPS tubes
  • Definition 11 - Supplementary data used by the analysis
  • Definition 13 - Wave 2D spectra direction and frequency
  • Definition 14 - Brightness temperature
  • Definition 15 - Seasonal forecast data
  • Definition 16 - Seasonal forecast monthly mean data
  • Definition 17 - Surface temperature or sea-ice data
  • Definition 18 - Multi-analysis ensemble data
  • Definition 19 - Extreme forecast index data
  • Definition 20 - 4D variational increments
  • Definition 21 - Sensitive area predictions
  • Definition 22 - Coupled atmospheric, wave and ocean models (with hindcast support)
  • Definition 23 - Coupled atmospheric, wave and ocean means (with hindcast support)
  • Definition 24 - Satellite image simulation
  • Definition 25 - 4DVar model errors
  • Definition 26 - MARS labelling or ensemble forecast data (with hindcast support)
  • Definition 27 - Forecasting Systems with Variable Resolution (Obsolete)
  • Definition 30 - Forecasting Systems with Variable Resolution
  • Definition 31 - EUROSIP products
  • Definition 50 - Member State data
  • Definition 190 -Multiple ECMWF local definitions
  • Definition 191 - Free format data descriptor data
  • Definition 192 - Multiple ECMWF local definitions
